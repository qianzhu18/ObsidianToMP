# ObsidianToMP v1.0.0

首发版本目标：在 Obsidian 本地完成公众号写作、预览、复制、发草稿全链路。

## 亮点
- 复制单入口：`复制到公众号`
- 复制时自动上传本地/内嵌图片到 S3 兼容图床
- 在线图片自动跳过，不重复上传
- 保留公众号草稿同步能力
- 多主题 + 多端预览（手机/平板/桌面）

## 品牌与链接
- 帮助入口统一到仓库：`https://github.com/qianzhu18/ObsidianToMP`
