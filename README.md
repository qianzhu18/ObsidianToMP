# ObsidianToMP

本仓库用于维护一个基于开源项目二次开发的 Obsidian 微信公众号发布插件。

## 目录结构
- `plugin/`: 可直接用于 Obsidian 的插件代码（包含 `manifest.json`, `main.js`, `styles.css`）

## 当前状态
- 以 `note-to-mp@2.0.0` 为原型进行本地优先改造
- 聚焦公众号单平台 MVP：预览、复制、发草稿
- 已集成额外主题并移除对付费授权链路的强依赖
